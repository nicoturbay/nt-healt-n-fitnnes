# NT Health & Fitness

A personal health and fitness tracking web app built with React + Vite + Tailwind CSS.

## Features

- **Today's Workout** -- Log exercises, sets, and reps. Load from preset templates or build custom.
- **Exercise Log** -- Full history of every workout, expandable with set detail.
- **Nutrition Tracker** -- Log meals with macros, set daily goals, track calories and P/C/F.
- **Progress Metrics** -- Body weight log with trend chart. Personal records tracker with best lifts.
- **Photo Log** -- Upload progress photos grouped by date with lightbox viewer.

All data is stored locally in the browser (localStorage). No backend, no account required.

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- React Router 6
- Recharts
- Lucide React icons

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

## Deploy

Connect to Vercel or any static host. Point root to this directory.
